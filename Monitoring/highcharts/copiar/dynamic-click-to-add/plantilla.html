Re: Highcharts and Joomla

by poulet1212 » Fri Aug 06, 2010 12:18 pm

Hi... It's not that easy. I use Joomla 1.5 and I'm developping a component in which I included highcharts.
First problem encountred: Highcharts needed some FX libs that Joomla 1.5 (mootools 1.12) doesn't have. Bit Mootools 1.2 does. Well... I give you some possibilities to get it running:


1) Use Joomla 1,5 with Mootools 1.2
There is many threads about this in the Joomla community, but it's likely to fail because even if some adapters exist, it will break some functionalities (Joomla menus or whatever...). I do not suggest it.

2) Another way is to Include also jquery lib, like you tried. I've used: http://ajax.googleapis.com/ajax/libs/jq ... ery.min.js
you absolutely have to edit this file and add "jQuery.noConflict();" add the end of this file. Otherwise you get some errors. This worked for me.

3) You could also try to open a new window where you output your graphs, without using Mootools at all. But your graphs are not included in a Joomla article.

Hope this helps.
